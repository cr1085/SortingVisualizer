# SortingVisualizer

¡Bienvenido a SortingVisualizer! 

Pequeño y elegante script en Java que demuestra visualmente (o mediante logs) distintos algoritmos de ordenamiento. Ideal para estudiantes, demostraciones rápidas y para integrarlo en proyectos educativos.

**Características**
- Implementación compacta en un único archivo: `SortingVisualizer.java`.
- Fácil de compilar y ejecutar con JDK (recomendado: Java 21 LTS).
- Código claro y comentado para aprender y modificar.

**Requisitos**
- Java Development Kit (JDK) 21 (LTS) recomendado.

**Cómo ejecutar (Windows - PowerShell)**
1. Si instalaste JDK 21 en `C:\Users\Pro\.jdk\jdk-21.0.8\bin`, puedes establecer temporalmente `JAVA_HOME` así:

```powershell
$env:JAVA_HOME = 'C:\Users\Pro\.jdk\jdk-21.0.8'
$env:Path = "$env:JAVA_HOME\bin;$env:Path"
```

2. Compila y ejecuta:

```powershell
javac SortingVisualizer.java
java SortingVisualizer
```

Si no quieres exportar `JAVA_HOME`, puedes usar la ruta completa al binario `javac` y `java`:

```powershell
C:\Users\Pro\.jdk\jdk-21.0.8\bin\javac SortingVisualizer.java
C:\Users\Pro\.jdk\jdk-21.0.8\bin\java SortingVisualizer
```

**Ejemplos de uso**
- Ejecuta sin argumentos para ver la demo por defecto.
- Modifica el `main` para probar diferentes entradas, tamaños de arrays o algoritmos.

**¿Quieres un JAR ejecutable o integración con Maven/Gradle?**
Puedo convertir esto en un proyecto con `pom.xml` o `build.gradle`, empaquetarlo en JAR ejecutable, o añadir una GUI mínima. Dime cuál prefieres.
